기술 스택

세부 기술

1. CI/CD

  • CI
    • eslint, prettier, husky
    • github action: lint-checker
  • CD
    • main 브랜치에 병합 시 자동으로 작동
    • 동적으로 .env 파일을 생성해서 api 키 등 중요한 요소를 보호
    • 리액트 빌드 후 S3에 자동 업로드 → 즉시 배포
    • cloudfront 캐시 무효화 설정

2. 배포

  • s3에 빌드된 리액트 배포 ⇒ s3의 정적 웹사이트 기능을 통해 자동 배포
  • 도메인, DNS 서버는 route53에서 구입
  • ACM에서 https 인증서 발급, cloudfront 도입
    • cloudfront를 통한 캐싱
    • cloudfront는 route53 뒤에 두어 도메인이 resolve된 다음에 접근

3. ChatGPT API

  • 첫번째 api
    • 사진을 넣으면 사진에 포함된 기호 중 데이터베이스의 정보와 가장 유사한 것들을 찾아 응답
    • base64 encoded image로 요청
    • 프롬프팅
      • 내부적으로 영어로 질의
      • 100여개의 기호에 대해 그 이름과 생김새를 텍스트로 정리
      • 다리미의 모양을 잘 파악하지 못함
  • 두 번째 api
    • 옷의 종류 및 세 가지의 소재와 그 비율을 입력하면, 그 관리법 및 관리법에 해당하는 세탁 기호를 정해진 데이터베이스 내에서 응답
    • 세탁 기호 응답은 첫번째 api와 비슷한 로직으로 구현
  • 세 번째 api
    • 의류 관련 질문을 입력하면 관리 방법을 응답
    • 단순 줄글 응답, gpt api 사용
